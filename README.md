## System Requirements

- **[NodeJS](https://nodejs.org/en/)**
- **[Yarn](https://yarnpkg.com/en/)**

## Installing

- `yarn install`

## Running Application

- `yarn serve`

## Other command

- `yarn release` # publish to master branch and publish portifolio
- `yarn build`
- `yarn push`
- `yarn watch`
- `yarn lint`
